# Pet-проект "SEO system"

## :grey_question: Описание проекта
Адаптивная страница компании продвижения бизнеса.

## :computer: Технологический стек
HTML, CSS, JavaScript

## :books: Практические умения
- создание таймера обратного отсчета
- создание интерактивных карточек для просмотра информации с обеих сторон
- создание анимаций переключения карточек

## :file_folder: Структура проекта
1. `fonts` - каталог с шрифтами.
2. `images` - каталог с картинками.
4. `styles` - стили.
6. `index.html` - файл для запуска программы.
